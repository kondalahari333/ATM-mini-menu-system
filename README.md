# ATM-mini-menu-system
##
This java programme simulates a basic ATM system that allows user to perform common banking operations through a menu-driven console interface.
##concepts used
*java basics
*conditional statements
*switch case
*user input
##menu options
1.check balance
2.Deposit amount
3.withdrawl amount
4.Exit
##inputs
*intial account balance
*user menu choice
*Deposit/withdrawl amount
##output
*current balance
*Transaction status message
*Exit confirmation
